# Gerador de Procuração

App estático para GitHub Pages que gera procurações em PDF no modelo Gregório & Morais.

## Modos

- Normal.
- Menor de 16 anos, representado por responsável.
- Maior de 16 anos, assistido por responsável.

Campos vazios são omitidos da qualificação no PDF. O rascunho é salvo apenas no navegador.

## Execução

Abra `index.html` ou sirva a pasta com qualquer servidor estático. Não há etapa de build.
